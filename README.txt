КАК ЗАПУСТИТЬ САЙТ:

1. Убедитесь, что файл "scorpion.glb" лежит в папке /assets/
2. НЕ открывайте HTML-файлы двойным кликом!
3. Запустите сайт через локальный веб-сервер:

   Варианты:
   - VS Code + расширение "Live Server" → ПКМ по index.html → "Open with Live Server"
   - XAMPP / OpenServer → поместите папку в htdocs/www → откройте в браузере http://localhost/scorpion-site/
   - Python: откройте терминал в папке и введите: python -m http.server 8000

Без сервера 3D-модель НЕ ЗАГРУЗИТСЯ!
